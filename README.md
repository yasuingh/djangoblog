# DjangoBlog — Full-Featured Blog Application

A production-ready Django blog with every essential feature built in.

## Features
- User auth (register, login, logout, profile, password reset)
- Blog posts with rich text, tags, categories, cover images
- Comments with nested replies, likes
- Post bookmarks / saves
- Search (full-text)
- RSS/Atom feeds
- Sitemap
- SEO meta tags
- Newsletter subscription
- Admin panel with custom actions
- REST API (Django REST Framework)
- Pagination, social share buttons
- Dark/light mode toggle
- Responsive, editorial design
