About

What is DjangoBlog?

DjangoBlog is a fully-featured, open-source blog platform built with Django. It was crafted to demonstrate best practices in Django development while providing a polished reading experience.

Features

  • Rich text editing with CKEditor
  • Tags and categories
  • Nested comments with likes
  • Bookmarks and post likes
  • Full-text search
  • Newsletter subscriptions
  • RSS feed & Sitemap
  • REST API with DRF
  • Dark / light theme
